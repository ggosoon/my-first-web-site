HTML

HTML stands for Hyper Text Markup Language. It is the standard markup language used to create web pages. HTML defines the content and structure of web content, and is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript 1.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page.
