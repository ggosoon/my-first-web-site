CSS

CSS stands for Cascading Style Sheets. It is a style sheet language used for describing the presentation of a document written in HTML or XML 1. CSS describes how HTML elements are to be displayed on screen, paper, or in other media 1. CSS separates the presentation of a document from its content, making it easier to maintain and update the document 1. CSS is used to style web pages, including the layout, colors, and fonts.
